PROJECT DESRIPTION

Gender and Age detector

EternCop is a simple computer vision project for detecting the gender and age of a person by capturing the image of the face. The front end is completely developed using the Django framework and the backend is composed of python libraries including OpenCV and Keras. The project is strictly developed for learning purposes and not fit for production environment.
Note : Please click 'z' button to exit from web cam
